# Speech2Text 

A speech-to-text transcription project using OpenAI's Whisper model that converts
audio input into text format with high accuracy across multiple languages.

## Project Overview

This project demonstrates how to use OpenAI's Whisper model for automatic speech
recognition (ASR). The pipeline follows these steps:

1. **Install Whisper** — Set up the OpenAI Whisper library
2. **Import Whisper** — Load the Whisper module into the environment
3. **Load the Model** — Initialize Whisper at the base level
4. **Transcribe** — Initialize the result by transcribing audio using the model
5. **Store Output** — Save the final result in text format
